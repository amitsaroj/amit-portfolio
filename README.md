# Amit Saroj Abyss 2030 Portfolio

A polished Next.js portfolio built from the Stitch "Personal Portfolio Showcase" visual direction.

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
```

## Stack

- Next.js App Router
- React
- TypeScript
- CSS modules via global app stylesheet
- Local Stitch-exported visual assets
